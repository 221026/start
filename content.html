AWS Cloud Foundations Services:

These foundational services form the core building blocks of AWS, providing the essential resources that most applications require:

Amazon EC2 (Elastic Compute Cloud):

Definition: A web service that provides secure, resizable compute capacity in the cloud.

Use Case: Allows users to launch virtual servers and scale them up or down as needed.

Amazon S3 (Simple Storage Service):

Definition: Object storage service that offers industry-leading scalability, data availability, and security.

Use Case: Store and retrieve any amount of data, from anywhere on the web, for applications like backups and big data analytics.

Amazon RDS (Relational Database Service):

Definition: A managed service that makes it easy to set up, operate, and scale a relational database in the cloud.

Use Case: Automates time-consuming tasks such as hardware provisioning, database setup, and backups.
